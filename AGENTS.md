# zz

Repository navigation tool combining ghq + fzf + zellij.

## State

Two kinds of state only:
- Session name: `zz:github.com.user.repo`
- Absolute path: `/home/aster/workspace/github.com/user/repo`

No relative paths (e.g. `github.com/user/repo`) in internal logic.
